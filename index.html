How to Put Your Site Online (Free)

Don't worry if you've never done this before — it only takes a few minutes and it's completely free. Follow these steps:

1 Create a GitHub Account (skip if you have one)

Go to github.com and click Sign Up. It's free. Use any email — this is just where your website files will live.

2 Unzip Your Download

Find the ZIP file you just downloaded and extract/unzip it. You should see files like index.html, about.html, etc.

3 Create a New Repository
  1. • On GitHub, click the + button in the top right → New repository
  2. • Name it something like my-trooper-site (no spaces)
  3. • Keep it set to Public
  4. • Click Create repository
4 Upload Your Files
  1. • On your new empty repo page, click uploading an existing file (the blue link)
  2. • Drag ALL the files and folders from your unzipped folder into the upload area
  3. • Make sure you include the public folder too!
  4. • Scroll down and click Commit changes
5 Turn On GitHub Pages
  1. • In your repo, click the Settings tab (gear icon, top of page)
  2. • In the left sidebar, scroll down and click Pages
  3. • Under "Source", select Deploy from a branch
  4. • Under "Branch", change the dropdown from None to main
  5. • Click Save
6 Wait & Visit Your Site!

Give it 1-2 minutes, then refresh the Pages settings page. A green banner will appear with your site URL:

https://yourusername.github.io/my-trooper-site/

That's it — share this link with your garrison!

help Troubleshooting
  • Site shows a blank page? Make sure you uploaded index.html to the root of the repo, not inside a subfolder.
  • Images not showing? Confirm the public/ folder was uploaded with the images inside it.
  • Still not working? Check that GitHub Pages is enabled (Step 5) and wait a few minutes for it to build.
